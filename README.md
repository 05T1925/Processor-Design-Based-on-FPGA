# Processor-Design-Based-on-FPGA

基于 FPGA 开发板的处理器设计课程项目。

## 项目定位

本项目选择课程题目 B：基于 FPGA 开发板的处理器设计。目标是在 Minisys FPGA 实验板上实现一个 RV32I 子集 CPU，并在基础 CPU 和小型 SoC 可运行的基础上，扩展 MAC 自定义指令，完成点积加速演示与 PPA 分析。

## 硬件与工具

- 开发板：Minisys 实验板
- FPGA：Xilinx Artix-7 XC7A100T
- 扩展板：EES-329B，仅作为备用资料，不作为主线外设
- 开发环境：Vivado 2018.3
- 主线外设：LED、拨码开关、七段数码管，UART 作为后期进阶选项
